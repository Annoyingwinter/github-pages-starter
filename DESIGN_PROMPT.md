# MDX 公开研究与 FIELD / 100 提示词边界

## 核验结论

Marcelo Design X 已在 2026 年以 MDX 继续运营，公开定位聚焦沉浸式网站、3D、WebGL、动效品牌系统与制作级前端。其公开文章强调：沉浸感是空间设计、动效语言、实时渲染、界面清晰度和工程纪律共同形成的系统，而不是单个视觉功能。

Marcelo 的公开短视频同时明确提醒：成熟网站不是 “one prompt websites”，真实产品需要意图与研究。公开视频可能要求评论关键词获取工作流，但公开页面没有提供可核验的完整私有提示词。因此本仓库不复制、不猜测、不伪装获取任何付费或私发内容。

## 本项目采用的公开方法

- 先确定视觉论点，再选择技术。
- 用镜头、景深、构图和滚动顺序讲故事。
- 让动效承担理解与连续性，而不是装饰。
- 为文字保留清晰区，让影像移动时界面仍可读。
- 从项目开始就设置移动端、性能与降级规则。
- 让设计与实现保持同一套决策，不在开发阶段丢失方向。

## FIELD / 100 的原创转译

本项目没有复制 MDX 的视觉身份。它从素材本身建立四幕：建筑外部、建筑内部、人物合照、参访现场；核心形式是“100 个证据构成一次真实抵达”。黑、矿物纸白、群青与信号橙的颜色系统，`FIELD / 100` 命名，中文文案，分区结构，横向建筑序列，人物长卷和分区档案均为本项目原创。
