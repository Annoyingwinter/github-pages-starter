# 龙城科创社会实践

北京科技大学同学 2026 年暑期赴常州开展社会实践的交互式影像记录，直接部署于 GitHub Pages。

页面不再展示“建筑外部”章节，重点改为讲清楚这次社会实践去了哪里、看了什么、为什么去。内容分区如下：

- 实践介绍：龙城科创学院背景、实践目的与三段参访路线。
- 参访现场：39 件学院、展陈与创新空间影像。
- 同学们：10 件参访过程与合影。
- 实践记录：23 件观看、讲解、操作与交流影像。

页面共展示 72 件素材。28 件建筑外部素材仍保留在 `assets/media/exterior/` 和数据清单中，但不再进入页面叙事，便于日后恢复或另作专题。所有原始文件继续保留；页面优先加载约 8.3 MB 的 WebP 预览图，只有打开查看器时才加载原始图片或视频。

## 内容依据

实践经过根据用户提供的现场素材整理。龙城科创学院背景参考常州科教城、常州市政府及相关高校的官方公开资料；页面正文不虚构团队名称、行程日期、合作关系或实践成果。

## 视觉方向

页面采用“日光档案室”方向：米白纸面、宋体中文、信号红、鼠尾草绿、暖黄与深绿形成不同内容区域。首屏从同学、学院内部和参访现场三个视角展开；正文通过大段编辑式排版、三站路线和独立影像章节，让文字与素材互相解释，避免把所有图片集中成相册。

## 文件

- `index.html`：社会实践叙事、三段路线、三个影像章节、资料来源与 SEO 元数据。
- `styles.css`：基础视觉系统、影像网格、响应式和 reduced-motion 降级。
- `field-v2.css`：当前“日光档案室”艺术方向及社会实践内容组件。
- `script.js`：可见素材筛选、分区画廊、原图/视频查看器。
- `media-data.js`：100 个原始素材的数据清单；前端过滤外部素材，仅展示 72 件。
- `assets/media-manifest.json`：可读的素材元数据。
- `assets/media/`：100 个原始文件、视频封面和 WebP 预览。

旧版 Pressure Fold 保存在本地 `_archive/pressure-fold-2026-07-16/`，并由 Git 忽略；仓库历史也可恢复此前版本。
